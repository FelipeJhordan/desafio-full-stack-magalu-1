## Rodando Aplicação
com yarn:
```BASH
yarn install
yarn dev
```
com npm:
```BASH
npm install
npm run dev
```
## Rodando Testes
com yarn:
```BASH
yarn test
```
com npm:
```BASH
npm run test
```
